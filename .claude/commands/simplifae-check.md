# simplifae-check

Run focused Simplifae regression and syntax checks.

```bash
node tools/simplifae-ingestion/simplifae.js check
```

Use after editing:

- `placsp_pipeline.py`
- `quality_gate.js`
- `packet_manifest.js`
- `promote_live_publicada_premium.js`
- semantic/parity validators
- Simplifae ingestion tests

This command does not mutate the prototype.
