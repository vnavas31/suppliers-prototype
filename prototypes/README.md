## Prototype Sources

This folder contains the editable HTML sources for the public prototypes.

- `tender-detail-v1/`: current Tender Detail source
- `discovery-v1/`: current Discovery recommendation source
- `simple-prototype/`: simplified comparison prototype

Recommended workflow:

1. Edit prototypes here on a feature branch.
2. Review changes in GitHub.
3. When approved, copy the final HTML to the `gh-pages` publishing branch.

The `gh-pages` branch is publish-only and should stay separate from source iteration.
